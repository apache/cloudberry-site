# pg_am

`pg_catalog` 模式中的 `pg_am` 表用于存储索引访问方法的相关信息。系统支持的每种索引访问方法都对应表中的一行记录。

| 列名      | 类型   | 引用 | 描述 |
|-----------|--------|------|------|
| `oid`     | oid    |      | 行标识符（隐藏属性；需显式选择） |
| `amname`  | name   |      | 访问方法的名称 |
| `amhandler` | regproc |      | 负责提供访问方法信息的处理函数的 OID |
| `amtype`  | char   |      | `t` 表示表（包括物化视图），`i` 表示索引 |
