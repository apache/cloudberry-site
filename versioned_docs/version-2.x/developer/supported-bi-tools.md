# Supported BI Tools

Any BI tool that supports Greenplum is also compatible with Apache Cloudberry. This includes, but is not limited to:

- Tableau
- Power BI
- Looker
- Apache Superset
- Metabase
- Qlik Sense

To set up and configure your preferred BI tool, follow the connection methods of [JDBC Driver](./use-jdbc-driver.md) and [ODBC Driver](./use-odbc-driver.md).

For further assistance, consult the documentation of your BI tool.
